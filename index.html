Capability Calculator [Java/HTML]

This is a Capability Calculator designed to analyze process data and generate key statistics, which are then automatically transferred to an Excel spreadsheet for easy download. I use it to evaluate both the potential capability of a process—assuming it’s perfectly centered between its specification limits—and its actual performance by measuring how consistently the process stays within those limits, considering its current average and variation. This helps determine how well a process is performing and whether it meets quality standards.
Tech stack: Java, Apache POI, Java Swing, Spring Boot
Highlights: Custom UI for file selection, auto-detection of SPC columns, and export of results to a separate “Capability” sheet in Excel.
